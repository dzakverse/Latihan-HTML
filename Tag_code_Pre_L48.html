contoh property CSS target selector p : 


            p {
                color: yellow
                margin: 20px
                font-size: 24px
            }

  1. Tag pre (Preformatted Text) Tag ini bertugas menjaga format tulisan persis seperti yang kamu ketik di editor.
    • Spasi & Enter: Kalau kamu kasih 5 spasi atau 3 kali Enter di dalam pre, browser akan nampilin jumlah yang sama (biasanya browser mengabaikan spasi berlebih).
    • Font: Secara default menggunakan font monospace (lebar tiap hurufnya sama).
    • Fungsi: Menjaga struktur baris kode agar tidak berantakan.
  2. Tag code (Computer Code) Tag ini bertugas memberi tahu browser bahwa teks di dalamnya adalah kode komputer (HTML, CSS, Javascript, dll).
    • Makna: Ini penting untuk SEO dan aksesibilitas agar mesin pencari tahu itu bukan teks obrolan biasa.
    • Font: Sama seperti pre, biasanya otomatis pakai font monospace.
    • Fungsi: Menandai potongan kode (bisa di tengah kalimat).
